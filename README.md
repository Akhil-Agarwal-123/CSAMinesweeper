## Features
- Board types:
  - Square mode (original)
  - Hexagonal mode
  - Triangular mode
- Game modes:
  - Original
  - Anti-Flag
  - Crazy House
  - Mine Tick
  - Automated Board
  - Climate Change
  - Build A Wall
  - Atomic
- Mine clustering algorithm
- Hints when stuck
- Resizeable GUI

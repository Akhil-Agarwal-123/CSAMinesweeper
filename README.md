An AP Computer Science A class project, made with Java swing.

## Features
- Board types:
  - Square mode (original)
  - Hexagonal mode
  - Triangular mode
- Game modes:
  - Original
  - Anti-Flag
  - Crazy House
  - Mine Tick
  - Automated Board
  - Climate Change
- Mine clustering algorithm
- Hints when stuck
